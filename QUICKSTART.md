## Need Help?

**Issue: Yarn not found**
```bash
npm install -g yarn
```

**Issue: QR code not scanning**
```bash
yarn start --tunnel
```

**Issue: App not updating**
- Press 'r' in terminal to reload
- OR shake phone and tap "Reload"

**Issue: Blank screen**
```bash
yarn start -c
```
